# Fullhouse Website

Static bilingual corporate website for Fullhouse / Фулхаус.

## Structure

- Russian pages are in the repository root.
- English pages are in `/en/`.
- Shared styles and scripts are in `/assets/`.

## Local Preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Deploy

The project can be deployed as a static site to Cloudflare Pages, GitHub Pages, Netlify, Vercel, or any regular web server. No build step is required.
